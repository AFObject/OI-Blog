递归实现指数型枚举

0x0201Easy-递归搜索

bool c[21];
void recursive(int t) {
	if (t == n + 1) {
		for (int i = 1; i <= n; ++i)
			if (c[i])
				printf("%d ", i);
		putchar('\n');
		return;
	}
	c[t] = true;
	recursive(t + 1);
	c[t] = false;
	recursive(t + 1);
}

void recursive(int t, int sel) {
	if (sel > m || sel + n - t + 1 < m)
		return;
	if (t == n + 1) {
		for (int i = 1; i <= n; ++i)
			if (c[i])
				printf("%d ", i);
		putchar('\n');
		return;
	}
	c[t] = true;
	recursive(t + 1, sel + 1);
	c[t] = false;
	recursive(t + 1, sel);
}

费解的开关

0x0204Medium结论贪心

\(5\times 5\) 的 01 矩阵中，点击一个位置，它及它上下左右格内的数都会变化，最少多少次点击变成全 0？

算法

考虑特殊情况： 假如不能动第一行，则对于第一行的任意一个数，如果它是 1，那么只能点击它下面的那个格子。

又有：点击的顺序不影响最终的结果。 一个格子被翻转的次数等于它四周的格子被点按次数之和。所以可以以任意顺序考虑点击的位置。

因此只用枚举第一行的 \(2^5\) 种状态即可。后面的方案都是确定的。假如最后不都为 0 那么这种做法就是寄的，继续搜索求解。

实现

第一次在 OI 里写异常处理。

int T;
int m[7][7];
int t[7][7];

int cnt = 0;
void doOper(int x, int y) {
	t[x][y] ^= 1;
	t[x - 1][y] ^= 1;
	t[x + 1][y] ^= 1;
	t[x][y - 1] ^= 1;
	t[x][y + 1] ^= 1;
	++cnt;
	if (cnt > 6)
		throw -1;
}

int main() {
	scanf("%d", &T);
	while (T--) {
		for (int i = 1; i <= 5; ++i)
			for (int j = 1; j <= 5; ++j)
				scanf("%1d", &m[i][j]);
		int ans = 0x3f3f3f3f;
		for (int s = 0; s < 32; ++s) {
			cnt = 0;
			try {
				for (int i = 1; i <= 5; ++i)
					for (int j = 1; j <= 5; ++j)
						t[i][j] = m[i][j];
				for (int i = 1, s0 = s; s0; s0 >>= 1, ++i)
					if (s0 & 1)
						doOper(1, i);
				for (int r = 2; r <= 5; ++r) {
					for (int c = 1; c <= 5; ++c)
						if (!t[r - 1][c])
							doOper(r, c);
				}
			} catch (int e) {
				continue;
			}
			int sum = 1;
			for (int i = 1; i <= 5; ++i)
				sum &= t[5][i];
			if (sum)
				ans = std::min(ans, cnt);
		}
		printf("%d\n", (ans == 0x3f3f3f3f) ? -1 : ans);
	}
	return 0;
}

Strange Towers of Hanoi

0x0205Medium递推DP

求 \(n\) 盘 4 塔的汉诺塔步数。

算法

考虑 \(n\) 盘 3 塔：

  • \(n-1\) 盘：A \(\to\) B
  • 第 \(n\) 盘：A \(\to\) C
  • \(n-1\) 盘：B \(\to\) C

因此 \(f_3(n)=2 \times f_3(n-1) + 1\)，其中 \(f_3(1)=1\)。可以知道 \(f_3(n)=2^n-1\)。

再考虑 \(n\) 盘 4 塔：

\[f_4(n) \le \min_{1 \le i \lt n} \{2 \times f_4(i) + f_3(n-i)\} \]

注意此处使用小于等于号。

参照具体数学上证明等于的方法：

我们迟早都必须要移动最下面的那 \(n-i\) 个盘子移动到 D 柱，当我们这么做的时候，上面 \(i\) 个必须已经在某个柱子上。而把下面的这 \(n-i\) 个盘子摆好以后，又要用 \(f_4(i)\) 次移动将上面的盘子放置在它上方（胡言乱语 /tuu）。因此至少需要有 \(\min_{1 \le i \lt n} \{2 \times f_4(i) + f_3(n-i)\}\) 次移动。

\[\therefore f_4(n) = \min_{1 \le i \lt n} \{2 \times f_4(i) + f_3(n-i)\} \]

实现

#define f3(x) ((1 << (x)) - 1)

int f4[13];

int main() {
    f4[1] = 1;
    for (int x = 2; x <= 12; ++x) {
    	f4[x] = 0x3f3f3f3f;
    	for (int i = 1; i < x; ++i)
    		f4[x] = std::min(f4[x], 2 * f4[i] + f3(x - i));
    }
}
