Sunscreen

0x0701USACOEasy+贪心排序

有 \(n\) 头牛，每头牛能忍受一个区间范围的阳光，然后有 \(m\) 瓶防晒霜，每瓶防晒霜能让一头牛身上的阳光变成一个定值，现在最大化能忍受阳光的奶牛数。

算法

我们的贪心策略一定是对于每一头牛：

  • 尽量能匹配到它的防晒霜
  • 在有多个匹配时选择最不影响其他牛的匹配
  • 如果非得因为这头牛然后让其他牛寄，也没问题，因为每头牛的权值都是 1。

对于第二条，考虑给高高在上的牛上牛就匹配值高的防晒霜，把值小的防晒霜留给下面的牛用（没被上面的牛用的一定能满足下面的牛的最小值，如果超过了它的最大值只能怪它自己，我已经把尽量小的留给你了）。

这种策略事正确的。

实现

将牛按区间起始值排序，防晒霜按值排序，均从大到小。每一头你从防晒霜中找第一个能被它用的，然后下一个。

int n, m;
std::pair<int, int> cow[100001], sun[100001];

int main() {
    n = read(), m = read();
    for (int i = 1; i <= n; ++i)
        cow[i].first = read(), cow[i].second = read();
    for (int i = 1; i <= m; ++i)
        sun[i].first = read(), sun[i].second = read();
    std::sort(cow + 1, cow + n + 1, std::greater<std::pair<int, int>>());
    std::sort(sun + 1, sun + m + 1, std::greater<std::pair<int, int>>());
    int ans = 0;
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            if (sun[j].second && cow[i].first <= sun[j].first && sun[j].first <= cow[i].second) {
                ++ans;
                --sun[j].second;
                break;
            }
    printf("%d\n", ans);
    return 0;
}

Stall Reservations

0x0702USACOEasy贪心排序

算法

萌萌题黄够了吧，和 廊桥分配 一个套路。

实现

区间有重复用 std::multiset。

int n;
struct Cow {
    int id, start, end;
    bool operator < (const Cow &rhs) const {
        if (start == rhs.start) {
            if (end == rhs.end)
                return id < rhs.id;
            return end < rhs.end;
        }
        return start < rhs.start;
    }
};
std::multiset<Cow> s;
int ans[500001];

int main() {
    n = read();
    for (int i = 1; i <= n; ++i) {
        int a = read(), b = read();
        s.insert({i, a, b});
    }
    int idx = 0;
    while (s.size()) {
        auto f = *s.begin();
        s.erase(s.begin());
        ans[f.id] = ++idx;
        int end = f.end;
        auto it = s.lower_bound({0, end + 1, 0});
        while (it != s.end()) {
            ans[it->id] = idx;
            end = it->end;
            s.erase(it);
            it = s.lower_bound({0, end + 1, 0});
        }
    }
    printf("%d\n", idx);
    for (int i = 1; i <= n; ++i)
        printf("%d\n", ans[i]);
    return 0;
}

Radar Installation

0x0703Medium-贪心计算几何

算法

从左到右考虑每一个点。对于每一个点，找到一个最右侧的能包含它圆。如果下一个点在先前所列的某个圆的范围内，包含进去。否则，/remake。


但这种方案还要再细化。如上图，「包含」关系时，就应该更新圆心往里缩一点，「继承」关系维持不变即可，「脱离」关系自然答案就应该加一。

实现

#define int long long
#define AVA 100000

int n, d;

struct Pos {
    int x, y, l, r;
    bool operator < (const Pos &rhs) const {
        if (l == rhs.l)
            return r < rhs.r;
        return l < rhs.l;
    }
} p[100001];

int dist(Pos a, Pos b) {
    return (a.x - b.x) * (a.x - b.x) + (a.y - b.y) * (a.y - b.y);
}

signed main() {
    n = read(), d = read() * AVA;
    for (int i = 1; i <= n; ++i) {
        p[i].x = read() * AVA, p[i].y = read() * AVA;
        const int k = (int)sqrt(d * d - p[i].y * p[i].y);
        p[i].l = p[i].x - k, p[i].r = p[i].x + k;
    }
    std::sort(p + 1, p + n + 1);
    int ans = 0, pos = -100000000000;
    for (int i = 1; i <= n; ++i) {
        if (p[i].y > d) {
            puts("-1");
            return 0;
        }
        if (p[i].l > pos) ++ans, pos = p[i].r;
        else pos = std::min(pos, p[i].r);
    }
    printf("%lld\n", ans);
    return 0;
}
